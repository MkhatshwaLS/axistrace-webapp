# axistrace-webapp
A full-stack project tracking application built with ASP.NET Core Web API and React/TypeScript. It uses Docker containers, secure JWT authentication, and REST APIs to help users track projects, milestones and progress in a simple and organized way.

# Overview
AxisTrace is a full-stack web application designed to improve transparency and communication in construction and architectural projects.

It allows clients and stakeholders to track project progress through clearly defined milestones, giving real-time visibility into each stage of the workflow. Instead of relying on manual updates, AxisTrace provides a centralized dashboard where progress, tasks, and updates can be monitored in one place.

The system includes features such as project and milestone tracking, role-based access for different users, secure authentication, and a responsive interface for easy access across devices.

Overall, AxisTrace focuses on reducing communication gaps, improving accountability, and bringing a more structured, digital approach to managing construction project progress.
